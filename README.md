# NVIM-ALERT

My personal NVIM settings

Built from scratch with the help of the best Primeagen :)

## Commands Cheatsheet

##### Run Netrw Listing

```
<Space>pv
```

#### Search Commands
```
<Space>pf # File search:wq
<Space>ps # Search for word in files
<Ctrl> + <p> # Repository search
```

#### Packer commands

```
:so # Reload file
:PackerSync # opens Packer + Updates plugins
```
